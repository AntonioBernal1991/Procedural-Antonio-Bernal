Generador 3D de Mapas Modulares y Caminos

Este proyecto implementa un sistema para generar mapas 3D modulares con caminos creados proceduralmente, optimizado para rendimiento y extensibilidad. El generador crea módulos usando cubos, genera caminos a través de ellos y gestiona los objetos de manera eficiente mediante un sistema de pooling. También incluye mecanismos para combinar mallas y generar caminos ramificados.

Características

  • Generación Procedural de Caminos: Genera caminos a través de módulos 3D basados en cubos, evitando repeticiones y permitiendo ramificaciones.
  • Object Pooling: Maneja instancias de cubos eficientemente para reducir la sobrecarga de memoria.
  • Combinación de Mallas: Optimiza el rendimiento combinando mallas, mejorando los FPS.
  • Creación Recursiva de Módulos: Crea dinámicamente módulos y los posiciona en el espacio 3D.
  • Control de Cámara: Proporciona un sistema básico de movimiento con teclas WASD para explorar el entorno generado.

Descripción del Código

Clases Principales

  • MapGenerator3D: Actúa como el nodo central, coordinando la creación de módulos y caminos.
    • DecideNextModulePosition: Determina la posición del próximo módulo y los puntos de salida del camino.
    • GenerateModules: Inicia la generación recursiva de módulos.
  • ModuleGenerator: Gestiona la creación de módulos individuales, incluyendo capas (suelo y hierba) y combinación de mallas.
  • PathGenerator: Genera caminos y gestiona ramificaciones.
  • ObjectPool: Implementa un sistema de pooling para optimizar la gestión de cubos.
  • MeshCombiner: Combina mallas por material para reducir llamadas de renderizado y mejorar FPS.

Configuración y Uso

Requisitos Previos

  • Unity versión 2020 o superior.
  • Proyecto configurado como 3D.

Instalación

  1. Clona o descarga el repositorio en tu proyecto de Unity.
  2. Coloca todos los scripts en la carpeta Assets/Scripts.
  3. Asigna los prefabs y materiales necesarios en el Editor de Unity.

Optimización de Rendimiento

  • Combinación de Mallas: La clase MeshCombiner reduce las llamadas de renderizado.
  • Object Pooling: La clase ObjectPool minimiza la creación de objetos en tiempo de ejecución.

Ejemplo de Código


MapGenerator3D.Instance.DecideNextModulePosition(5, 5, CurrentDirection.RIGHT);
StartCoroutine(ModuleGenerator.StartRecursiveGeneration(10));
